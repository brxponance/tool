## Using Idea Screening Hot Keys Page 17198

Launch it with FactSet Search: @ISC

Idea Screening offers hot key functions for [editing screen results](https://my.apps.factset.com/oa/pages/pages/16699#results1) using your keyboard. The following tables show the functions that are available and their associated hot key combinations.

### Creating or Editing Parameters

|**Function**|**Hot Key**|
|---|---|
|Navigate through columns|**UP/DOWN arrow keys**|
|Select sequential columns|**SHIFT+UP/DOWN arrow keys**|
|Add a column|**ENTER**|
|Remove a column|**DELETE**|

### Working with Results

|**Function**|**Hot Key**|
|---|---|
|Move focus from results to comps to pagination|**TAB**|
|Advance through options|**UP/DOWN arrow keys**|
|Search for a result|**CTRL+SHIFT+F**|
|Select an item|**SPACEBAR**|
|Add a result as a custom comp (in Company screens)|**ENTER**|
|Remove a result (when the row is highlighted)|**DELETE**|

### Working with Lookups

|**Function**|**Hot Key**|
|---|---|
|Expand/Collapse top-level nodes|**RIGHT/LEFT arrow keys**|
|Advance through options|**UP/DOWN arrow keys**|
|Open/close information panel|**SHIFT+RIGHT/LEFT arrow keys**|
|Select an item|**SPACEBAR**|
|Remove an item from the Selected box|**DELETE**|
