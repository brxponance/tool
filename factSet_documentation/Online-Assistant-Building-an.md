## Specifying Model Options

Use the Model Options window to create a new model or edit an existing model. To launch Model Options, click the **Model Options** button in the top right corner of any Alpha Testing report.

Alternatively, from the Workspace report, you can click on the pencil icon ![](online-assistant/26048.html) to the right of each model option (Universe, Time Series, Returns Sources, Factors List) to launch the Model Options window.

Once you enter all of the model options, click **Done**.

|**Note**|When you click **Done** to accept the changes made in Model Options, Alpha Testing will check that all of your inputs are valid. Alpha Testing will not accept changes to Model Options until all inputs are valid.
|---|---|
If errors are found, the ![](online-assistant/26048.1.html) icon will appear to the right of the Options tab that contains the error. To view more information about the error(s), click the **Errors Found** button at the top right of the Edit Model Options window.|

## Running your Model

To generate model results and reports, click the **Run Model** button in the top right corner of any Alpha Testing report.

Once a model has been run for the first time, you can use the drop-down menu to select from the following additional options for subsequent runs: Append Run, Full Run, Selective Dates, Schedule.

For more information on the model run options, see [Running a Model](https://my.apps.factset.com/oa/pages/pages/20830).

## Analyzing Model Results

Access the available Alpha Testing reports in the left-hand pane of the application. Several reports are available within the FactSet:Default report setting.

### Default reports

-   **Workspace**
    
    Display an overview of the [model options](https://my.apps.factset.com/oa/pages/20829#model_options) specified.
-   **Constituents**
    
    View a list of companies in your passing universe and statistical details for each.
-   **Benchmark**
    
    View a list of companies in your benchmark and statistical details for each.
-   **Periods**
    
    View summary statistics at the period level; use this report to view results over time.
-   **Fractiles**
    
    View summary statistics by factor fractile and analyze factor interactions.
-   **Factors**
    
    View detailed results for each factor.
-   **Multiple Factor Regression**
    
    View [ANOVA](https://my.apps.factset.com/oa/pages/pages/3920#anova) statistics for a multiple factor regression of your factors against your forward return for each period.
