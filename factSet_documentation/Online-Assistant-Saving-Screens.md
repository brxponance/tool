Click the **Save** button ![](online-assistant/20144.html) and select "Save Screen" to save your current screen to the [Screen Library](https://my.apps.factset.com/oa/pages/pages/16696#sl).

![](online-assistant/20144.1.html)

The different saving options are:

-   Save Screen: saves all your screening parameters. This allows you to run the same screen over time as situations and conditions change.
-   Save Report: saves the results of the screen. Because conditions are always changing, you may get different results the next time you run a screen with the same parameters.
-   Save Identifier: saves the list of passing companies to an OFDB file that you can use in other FactSet applications.

## Saving Report Settings

To save a copy of your report columns into a template, click the **Save** button from a results view and select "Save Report." After you save your report, you can bring the same set of columns into any other screen using the Report Library tab in the Edit Report dialog box.

If you would like your saved report to populate in future screens, select the "Use as Default Report" check box in the Save Report to Library dialog box, or the Details pane of the Open Report dialog box. You can assign a default report to each view in the application.

## Saving Identifier Lists

You can save the list of identifiers that pass your screen and use the identifier list within Idea Screening and other FactSet components.

To save an identifier list from the Results view in Idea Screening:

1.  Click the **Save** button from a results view, select "Save Identifiers," and then select the type of identifiers you want to keep.
2.  Select the directory where you want to save your identifier list in the left pane. Options available include: [Personal](https://my.apps.factset.com/oa/pages/pages/3935#personal_library), [Client](https://my.apps.factset.com/oa/pages/pages/3922#client_library), and [Super\_Client](https://my.apps.factset.com/oa/pages/pages/3938#super_client_library).
3.  Enter a name and description for your identifier list.
4.  Click the **Save** button to create your new identifier list.

If you are saving Custom Comps in a Company screen, you also have the option to save the custom comps, the screening results, or both.
