## Using a Custom Universe

Idea Screening calculates Volume Totals and League Tables based on your screen results. This excludes manual additions or deletions you have made when using the Volume Total or League Table functionality. To run a Volume Total or League Table based on a custom universe, use the following steps.

1.  Run a screen and use the Custom Comps section to define your universe. Add deals that passed your screen by right-clicking a row and selecting "Add to Comps" or searching for and adding one using the [Find/Add](https://my.apps.factset.com/oa/pages/pages/16699#find) button.
2.  Save the list by clicking the **Save Identifiers** button and using the "Save Custom Comps Identifiers" option.
    
    ![](online-assistant/20148.3.html)
    
3.  Open a new screen and add a DealID parameter to your workspace. Click the **Lookup** button to add the screen you saved from step 2.
4.  Run the screen using Volume Totals or League Tables.
