## Getting Started 

To get started, click the **Fund** button located at the top of the application [Home](https://my.apps.factset.com/oa/pages/pages/20595#home) page.

You can choose from the following options in the Suggested Screens section: 

-   Click the **New Fund Screen** ![](online-assistant/26763.html) button to create a blank report. All funds are included in the default universe. 
-   Click the **Starter Screen** button to get some basic result columns on the default fund universe. 
-   Click a predefined fund screen (e.g., European Funds) to open a report that has a FactSet-established universe and format. Each demonstrates the main features of the application and can serve as a building block for creating your own screens.

## Establishing the Starting Universe

The starting universe is the full list of securities that are available to you within the universe that you are working with.

To start limiting your universe, start typing in the Add Criteria search box, and then select the limit you want from the populated list of results.

## Managing Universe Limitations 

You can use criteria items as building blocks when creating a universe. Each item (i.e., limitation) is a specific rule that defines a set of securities. 

Screening criteria can be assigned in two ways – through the Add Criteria search box, or the **Browse for Criteria** ![](online-assistant/26763.1.html) button. 

![](online-assistant/26763.2.html)

### Searching for Limits with Typeahead 

Use the type-ahead search functionality to quickly find a criteria item. Enter all or part of it into the Add Criteria box to view a list of matching items. Use your cursor or keyboard arrows to locate and select your desired limit.

Most limit types are automatically added to your criteria list when you select them. However, formula and function limits need to be converted. To do this, simply add an operator (e.g., greater than one billion). For example, to limit to Morningstar U.S. funds with a total net asset value greater than one million: 

1.  Select the formula from the search results. 
2.  Add "`>1000000`" to the end of the formula.
    
    ![](online-assistant/26763.3.html)
    
3.  Press **ENTER**. 

### Browsing for Limits

To browse for universe limits, click the adjacent **Browse for Criteria** ![](online-assistant/26763.4.html) button and choose the Qualitative or Quantitative tab. 

The Qualitative tab contains categories of various criteria items, such as institutions, hedge funds, and domiciles. You can reveal and hide the underlying sections of each category by clicking the **Expand/Collapse** ![](online-assistant/26763.1.html) buttons.

![](online-assistant/26763.5.html)

Press the **CTRL** key and click to select more than one item, or hold down the **SHIFT** key and click to select sequential items. You can apply an entire category section as a limitation by double-clicking its name, or selecting it and clicking the **Add** button. 

The Quantitative tab can be used to add FactSet formulas or [formulas that you define](https://my.apps.factset.com/oa/pages/pages/20605#Managing).

![](online-assistant/26763.6.html)

Once a quantitative limit is selected, it is added to the Add Criteria box. Add an operator to convert it into a formula. 

To edit a formula limit, select it within the Criteria column of the Select Criteria pane. Apply your edits in the Add Criteria box and press **ENTER**. 

### Adding and Removing Limits 

To include an item, double-click or select it and click the **Add** button. Once an item has been selected, it appears in the Select Criteria pane. To remove a limit from your current limitations, click the **Remove Row** ![](online-assistant/26763.7.html) button to the right of the limitation, or right-click it and select "Delete." To remove all limitations, highlight any row, right-click, and select "Delete All."  

![](online-assistant/26763.8.html)
