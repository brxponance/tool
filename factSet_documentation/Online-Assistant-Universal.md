## Using Universal Screening (New) Page 20593

Launch it with FactSet Search: @US

Develop, test, and confirm investment strategies more efficiently with the new version of Universal Screening.

The application allows you to identify equities, fixed income securities, institutions, and funds that meet specific criteria. Access key statistics and values, historical fundamental data, and limitless customization – all in a single platform. Plus, you can choose from a vast library of predefined items, or create your own proprietary screening criteria based on the qualities you feel are most important.

This page highlights the new features and enhancements available in the new Universal Screening. 

![](online-assistant/25661.html)

## Highlighted Features

**Horizontal Multi-Pane Layout**

Assess your investment strategies more effectively and improve your research experience. The new multi-pane layout arranges all steps of the screening process in one, continuous view. Build an investable universe, manipulate the report content, and interpret results in a progressive manner for thousands of securities worldwide.

You can use the intuitive layout to manage all screening conditions and gain valuable insight into how each one influences the number of passable securities.

**Type Ahead**

Quickly locate criteria items and groupings, or build custom formulas using the new type-ahead search functionality. This timesaving component has been expanded from the previous Universe and Parameter views. It now offers predefined filters and searches beyond formulas to generate a list of relevant classifications, data items, functions, and more. The more filters and keywords you use, the finer your search results.

**Function Builder**

Combine functions with formulas in seconds. The new Function Builder tool guides you through the process of calculating logical, mathematical, statistical, growth, and other operations. Click the **Browse for Formulas** button ![](online-assistant/25661.1.html) or start typing within the Criteria or Results pane to get started. Once you have selected a function and specified its inputs, Function Builder will provide the correct syntax, which includes wrapping the function around an existing formula.

**Report Filtering**

Easily limit your view with column filters. These allow you to view the distribution of data across parameters and specify which universe data you want to show. You can set numeric or textual filters to remove outliers or analyze a subset of your overall results. When you apply numeric filters, the counts are updated to reflect how many data items pass the criteria you have selected.

**Parameter Categories**

Parameters in the Parameter view can now be organized into categories. By default, there are two – FROZEN\_COLUMNS, which contains Name and Symbol rows, and PARAMETERS, which contains all applied parameters. A new REPORT\_GROUPS\_SELECTION category at the bottom of the Parameter view shows all groupings that have been applied to your report.

## Error Messages and Debugging Information

**Unique Error ID for Tracking**

When you encounter a service error, a unique error ID is included in the error dialog. If reported, the Error ID can be used to trace to the exact instance in FactSet's error logs. It will help FactSet to quickly locate the issue in the logs and as a result reduce the response turnaround time.
