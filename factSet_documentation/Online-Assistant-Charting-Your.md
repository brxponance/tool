## Charting Your Results Page 21125

With Universal Screening, you can easily use your results to create various charts on the fly. Create multiple charts in a single screen and download them so that they are always available to you. These charts are tightly integrated with the application; each lets you visualize your data and simplify your analysis. 

Charts help you answer questions about your screen's [sector/industry distributions](https://my.apps.factset.com/oa/pages/pages/21125#Sector), how securities compare to the sector average, and [how closely prices and sales correlate to the securities](https://my.apps.factset.com/oa/pages/pages/21125#ClosingPriceSales). There is no limit to the number of charts you can add to Universal Screening.

## Accessing Charts

You can easily chart columns and groupings to view a graphical representation of your report. 

To access charts in Universal Screening: 

1.  Click the **Charting** button ![](online-assistant/26436.html) on the toolbar of the Results pane. 
    
    ![](online-assistant/26436.1.html) 
2.  Double-click or drag-and-drop a [chart type](https://my.apps.factset.com/oa/pages/pages/21125#Viewing) from the populated dashboard. If the dashboard does not appear, simply click the **Add Component** button ![](online-assistant/26436.2.html) to access it and the available chart options. These include Column, Bar, Donut, Top 10 / Bottom 10, Histogram, Scatter, and Heat Map. 
    
    ![](online-assistant/26436.3.html) 
3.  Depending on the chart type selected, you may need to specify between one and three items. To change the column, grouping, statistic, or value used in the chart, select an item from the corresponding drop-down menu(s) at the top of the tile. You can also search for one using the type ahead functionality. 
    
    ![](online-assistant/26436.4.html) 
4.  Click the **View All Panes** button ![](online-assistant/26436.5.html) to access the default application layout, which includes the Select Criteria and Format Result panes. Click the **Results** button to revert to the main Results table. 

|**Note**|By default, any filtering you select in the report will be applied to the charts.
|---|---|
Additionally, the N/A values shown on each chart may be mislabeled (e.g., misrepresented as a large negative number) if they are part of a text-based grouping such as FactSet/Sector.|

## Viewing Chart Types

The following chart types are available in Universal Screening. The different types allow you to quickly analyze your screen using visual cues. 

|**Chart Type**|**Description**|**Display**|
|---|---|---|
|Column Chart|Displays values in a vertical bar format. 
Required fields: Column, Group By|
|Bar Chart|Displays values in a horizontal bar format. 
Required fields: Column, Group By|
|Donut|Displays values in a donut format.  
Required field: Group By|
|Top 10 / Bottom 10|Displays the top 10 and bottom 10 values in two-colored vertical bar formats.
Required field: Column|
|Histogram|Displays values in a bar format. 
Required fields: Column, Number of Data Distribution Buckets|
|Scatter Plot|Displays values in a scatter plot format. 
Required fields: Column, Column|
|Heat Map|Displays values using color and scaled areas in a grid format. Securities with greater representation within the results are indicated by larger areas. Only one grouping level (i.e., group by sector or group by exchange) is supported.
Required fields: Group By, Column|

## Outputting Charts 

Universal Screening offers several output options for your charts. 

To download a chart to a PDF file: 

1.  Click the **Download Charts** button ![](online-assistant/26436.14.html) on the toolbar of the Results pane. 
2.  If you are in the FactSet Workstation, a Download dialog box will appear. Select "Open" to view the PDF file, or click "Save."
    
    If you are in FactSet Web, the file will appear at the bottom of your browser window and within your computer's Downloads folder. 
    

To copy a chart into a Microsoft Office application: 

1.  Click the top-right **Tools** menu ![](online-assistant/26436.15.html). 
2.  Select the "Copy Chart Image" option. 
    
    ![](online-assistant/26436.16.html) 
3.  If you are using Screening via the FactSet Workstation, right-click or press **CTRL+V** to paste the chart into Microsoft Word, PowerPoint, or Excel. 
    
    If you are using Screening via FactSet Web, right-click the chart, select "Copy Image," and then paste it into Microsoft Word, PowerPoint, or Excel.
    

|**Note**|Charts that are exported from Universal Screening are static; they cannot be refreshed.|
